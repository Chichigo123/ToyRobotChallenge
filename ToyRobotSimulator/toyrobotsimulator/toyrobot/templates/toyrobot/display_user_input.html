Welcome to Toy Robot Challenge

The robot must be initially placed on the 5 x 5 table

Command Description
PLACE x,y,f - where x=[0-{}), y=[0-{}), f(facing)={} e.g PLACE 0,1,EAST
MOVE - moves the robot one unit forward in the direction it is currently facing
LEFT|RIGHT - rotates the robot 90 degrees in the specified direction without changing robot's position
REPORT - announces the x y f of the robot

Issue the command separated by a new line. Checking of the validity of the commands is not yet fully implemented.
